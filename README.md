# AWS-Secure-Environment

## Overview
Built and secured a cloud-based server using AWS EC2, IAM, and Linux system configuration.

## What I Did
- Launched EC2 instance and connected via SSH using key-based authentication
- Restricted SSH access using security groups (limited to my IP)
- Created IAM users and applied least privilege principles
- Configured system logging and monitored authentication logs
- Analyzed login attempts and system activity using Linux log files

## Security Concepts Applied
- Least privilege
- Network access control
- Secure authentication (SSH keys)
- Log monitoring and analysis

## Tools Used
- AWS (EC2, IAM, Security Groups)
- Linux
- SSH
